Цель проекта

Создать веб проект собственных курсов.

Отличительные особенности проекта, дающие конкурентные преимущества

  1. Курсы будут обучать профессии всесторонне. Студент после успешного обучения  будет обладать достаточными знаниями и достаточной практикой для самостоятельного выполнения коммерческого проекта.
    1. Курс будет включать обучение всем необходимым в коммерческих проектах знаниям.
    2. Знания будут проверяться опросами, самостоятельными заданиями.
    3. Студенты будут проходить практику в реальных проектах.
    4. Студенты будут создавать собственные проекты.
    5. Студенты будут проходить стажировку в действующих проектах.
  2. После успешного окончания курса выпускники будут трудоустраиваться фрилансерами с возможностью работы в действующих аутсорсинговых агентствах.
  3. Часть курса (подкурсы/темы) будут бесплатными. Их также можно будет оплачивать для получения помощи от наставника.
