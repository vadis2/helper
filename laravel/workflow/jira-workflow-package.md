# JIRA workflow to create package
## Common part
````
 Planning
 Create branch
 Choose package name
 Create tree of directories
 Investigation
 Create specification
 Create composer.json
 Create package discovery
 Check and edit tree
 Create/Edit/Publish Service Provider
 Edit main composer.json
 Checking/Run composer dump-autoload
 Create entity block scheme
 Create logic scheme
 Evaluation
 Create tests
````
## Controller part
````
 Create routes
 Create controller and test action
 Create migration
 Create model
 Create seed
 Create links, menu
 Create index page view
 Delete demo action
 Create Add page view
 Create Request Validation
 Create store action
 Create Edit page view
 Create update action
 Create delete action
 Create publishing
````
## Facade part
````
 Create package class
 Create a facade
 Create tests
 Create publishing
````
## Final part
````
 Create config
 Publish package
 Install package
 Deployment
 Create Readme.md 
````
