# 4. Creating a Vue CLI project
````
cd dp
vue create hello-world
````
Options:
````
Babel, Router, Linter
history mode for router
basic linter
Lint on Save
package.json
No preset!
````
## Upgrade vue instance o vue 3 (this option exists for vue 2)
````
cd hello-world
vue add vue-next
````
Open new project in the IDE.

See package.json

## Start server
````
npm run serve
````
